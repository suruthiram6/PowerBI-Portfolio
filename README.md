 Indian Fashion E-commerce Analytics Dashboard

📌 Project Overview
This project analyzes Indian fashion e-commerce sales data using Microsoft Power BI. The dashboard provides insights into revenue, customer behavior, product performance, discounts, and sales trends through interactive visualizations.

🎯 Project Objectives
- Analyze sales and revenue trends
- Identify top-performing products
- Understand customer purchasing behavior
- Compare category-wise performance
- Evaluate discount impact on sales

🛠 Tools Used
- Microsoft Power BI
- Power Query
- Microsoft Excel

 📂 Project Files
- 📄 Project Report (PDF)
- 📊 Power BI Dashboard (.pbix)
- 🖼 Dashboard Screenshots

 📈 Dashboard Features
- Revenue Analysis
- Sales Trend Analysis
- Customer Insights
- Product Performance
- Category-wise Sales
- Interactive Filters and Slicers
